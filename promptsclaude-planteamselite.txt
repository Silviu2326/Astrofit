## 📝 NOTAS GENERALES

### Elementos Clave a Aplicar en TODOS los Módulos:

1. **Hero Section:**
   - Gradiente de 3 colores: `from-X via-Y to-Z`
   - Efectos blur orbs animados con `animate-pulse`
   - Patrón de fondo (grid o dots) con opacidad baja
   - Título con icono en container backdrop blur
   - Border `border-white/20` o `border-white/50`

2. **Cards:**
   - `rounded-3xl` (principal)
   - `bg-white/80 backdrop-blur-xl`
   - `border border-white/50`
   - Decoración blur orb en esquina
   - `shadow-xl hover:shadow-2xl`
   - Efecto shimmer en hover para interactivos

3. **Animaciones:**
   - Entrada: `initial={{ opacity: 0, y: 20 }} animate={{ opacity: 1, y: 0 }}`
   - Hover: `whileHover={{ scale: 1.03, y: -8 }}`
   - Stagger: usar `delay: index * 0.1` en listas

4. **Componentes:**
   - Inputs: `rounded-2xl` con `focus:ring-4 focus:ring-indigo-100`
   - Botones: gradientes con `group` y overlay hover
   - Badges: `rounded-full` con gradientes sutiles de fondo
   - Progress bars: gradientes de 3 colores con animación

5. **Iconos:**
   - Contenedor: `p-2 bg-white/20 rounded-xl backdrop-blur-sm`
   - Glow effect opcional: div absoluto con blur

### Paleta de Gradientes Sugerida por Tipo:

- **Analíticas/Dashboards:** `from-blue-600 via-indigo-600 to-purple-600`
- **Sensores/Tecnología:** `from-cyan-500 via-blue-500 to-indigo-500`
- **Tiempo Real/Live:** `from-emerald-500 via-teal-500 to-cyan-500`
- **Eventos/Competencias:** `from-purple-600 via-pink-600 to-rose-600`
- **Scouting/Evaluación:** `from-blue-500 via-indigo-500 to-purple-500`
- **Comparación/VS:** `from-red-500 via-purple-500 to-blue-500`
- **Proyecciones/Predicciones:** `from-orange-500 via-red-500 to-pink-500`

### Paleta de Badges por Estado:

- **Conectado/Activo:** `bg-green-500 text-white`
- **Desconectado/Inactivo:** `bg-gray-400 text-white`
- **Sincronizando/Proceso:** `bg-blue-500 text-white animate-pulse`
- **Advertencia:** `bg-yellow-500 text-white`
- **Crítico/Alerta:** `bg-red-500 text-white`
- **Excelente/Ganador:** `bg-gradient-to-r from-yellow-400 to-orange-500`

### Recordatorios:

- ✅ Siempre usar `relative z-10` en contenido sobre decoraciones
- ✅ Agregar `overflow-hidden` a contenedores con blur orbs
- ✅ Usar `group` y `group-hover:` para efectos hover complejos
- ✅ Mantener consistencia en espaciados (gap-6 entre cards, gap-4 elementos)
- ✅ Implementar responsive con `md:` y `lg:` breakpoints
- ✅ Para módulos de comparación, usar animaciones desde lados opuestos
- ✅ Para datos en tiempo real, usar `animate-pulse` en indicadores activos
- ✅ Para scouting, destacar ratings con progress circles/stars
- ❌ NO crear archivos nuevos
- ❌ NO eliminar funcionalidad existente
- ❌ NO cambiar la estructura de datos o lógica
