# 📝 NOTAS GENERALES PARA TODOS LOS PROMPTS

## ⚠️ IMPORTANTE:
1. **SIEMPRE** revisar `GUIA_ESTILOS_UI.md` antes de empezar
2. **NO CREAR NUEVOS ARCHIVOS** - Solo editar existentes
3. Usar **TypeScript** correctamente
4. Implementar **framer-motion** para animaciones
5. Incluir **Lucide React** para iconos
6. Datos **mockeados realistas**
7. **Responsive** completo (mobile-first)
8. **Código limpio** y bien comentado

## Checklist de Estilización:
- [ ] `rounded-3xl` para cards principales
- [ ] `bg-white/80 backdrop-blur-xl` para glassmorphism
- [ ] `border border-white/50` para definición
- [ ] Gradientes de 3 colores en headers
- [ ] Patrones sutiles (dots/grid) en backgrounds
- [ ] Animaciones de entrada con framer-motion
- [ ] `whileHover` para interactividad
- [ ] `shadow-xl` y `hover:shadow-2xl`
- [ ] Iconos en contenedores con backdrop blur
- [ ] Progress bars con gradientes animados

## Paleta de Colores Principales:
- **Green/Emerald/Teal**: Plantillas de dietas (natural, saludable)
- **Lime/Green/Emerald**: Listado de dietas (energía, vida)
- **Amber/Orange/Red**: Nueva dieta (creación, calor)
- **Yellow/Amber/Orange**: Editar dieta (ajuste, refinamiento)
